# SHINE LEADERSHIP ACADEMY 🌟

Welcome to the **Shine Leadership Academy** open-source repository! 

## 📖 About the Project
Shine Leadership Academy is an initiative dedicated to empowering African women through leadership training, mentorship, and community engagement. This project provides the static web platform for the academy, designed to be modern, fully responsive, and accessible.

## ✨ Features
- **Responsive Design**: Built with modern CSS (Tailwind CSS) ensuring the site looks great on all devices (mobile, tablet, desktop).
- **Mentorship Platform**: A dedicated section to connect mentors with aspiring leaders.
- **Events & Activities**: Stay updated with the latest workshops, webinars, and meetups.
- **Member Dashboard**: A personalized space for members to track their progress and interact with the community.

## 🏗️ Project Structure
The project is organized as a purely static website. No complex build tools are required to run it locally.
- `index.html` — The main landing page.
- `mentorship.html` — Mentorship details and registration section.
- `events.html` — Upcoming events and activities.
- `dashboard.html` — The member dashboard area.

## 🚀 Getting Started

Since this is a static HTML/CSS website, getting it running locally is incredibly simple:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/wenjus26/beproud-shine.git
   ```
2. **Open the project:**
   Navigate into the project directory:
   ```bash
   cd beproud-shine
   ```
3. **Run locally:**
   Simply double-click `index.html` to open it in your favorite web browser. 
   Alternatively, if you use VS Code, you can use the **Live Server** extension for hot-reloading.
